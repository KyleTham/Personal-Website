Database Project

Project for: CS 157A Database Management Systems

Overview

A team that included myself along with two others were assigned to create a database system where Users interact with the database by using certain functions. An idea was brainstormed by each of us, and we came up with three different ideas: I proposed we create a platform where users can buy games - basically like Steam, a digital distribution platform where users can purchase video games. My teammates suggested that we could create a platform that would be similar to Yelp, and a platform that would be similar to Amazon. We ended up designing the game distribution platform.

Requirements

  • A minimum of 5 tables that are relational to each other in some way.
  • 15 Distinct Functions for Users, excluding database admin functions.
  • At least one trigger involved in the database.

    • Features

    We created 6 total relational tables, which included:

    • User
    • Credit Card
    • Game
    • Review
    • Publisher
    • Owned Library
    Owned library is relational to User and Game to their respective ID's.
    Publisher is relational to game, where publisher is the publisher of the game.
    Review is relational to User and Game on their ID's.
    Game references publisher.
    User references Credit Card on their ID, name, and credit card number.


    For the 15 user functions, what we included and came together to create the functions users would most need:

    • Seeing the ratings of games in order from highest to lowest (5 being the highest, 1 being the lowest)
    • Sort games by price
    • Search a game by the name
    • Search games made by a certain publisher
    • Purchase a game
    • Check which games they own
    • Deposit money/funds into the platform
    • Insert their credit card info into their account
    • Reset their password
    • Create an account
    • Find the user who has bought the most games
    • Add a review to a game
    • Find a game that has no reviews yet
    • Edit a review that they have written
    • Deleting their reviews
